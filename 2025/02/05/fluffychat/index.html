fluffychat的安装与使用

2025-02-05T03:04:37.000Z2025-02-06T02:44:14.983Z6112 min

FluffyChat 是什么

FluffyChat 是一款基于 Matrix 协议 的开源、非营利即时通讯客户端，主打简洁易用和安全性，适用于 Android、iOS、Linux 等平台，并提供网页版。其核心特点如下：

  1. 去中心化与安全
    • 支持自选服务器或自托管，数据无需依赖中心化服务器，用户可完全控制通信隐私。
    • 默认启用端到端加密、跨设备验证（Emoji 验证）、加密备份，确保消息安全。
  2. 多功能通信
    • 支持文字、图片、文件、语音消息、位置共享，以及实验性的音视频通话功能。
    • 提供无限群组聊天、公开频道（支持数千人）和社区空间（Spaces），便于组织复杂对话。
  3. 美观与跨平台
    • 采用 Material You 设计语言，适配深色模式，界面简洁可爱，支持自定义主题和贴纸。
    • 兼容 Element、Nheko 等 Matrix 生态应用，实现多平台无缝切换。
  4. 开源与社区驱动
    • 代码完全开源，依赖社区贡献完善多语言支持和功能迭代，支持通过 Liberapay 捐赠项目。

注册教程（以移动端为例）

  1. 下载并安装 FluffyChat

    • Android：通过 F-Droid 或 Google Play 搜索下载。
    • iOS：在 App Store 搜索安装。

  2. 在浏览器打开https://web.synod.im/

    • 按照提示注册账户（如下图）

  3. 启动应用并选择服务器

    • 打开 FluffyChat，点击 “创建账户”。
    • 输入 Matrix 服务器地址（默认的 matrix.org在国内被墙了，可选择其他公共服务器如 synod.im，HPB在这个服务器有群）。

  4. 填写注册信息

    • 输入电子邮箱（即 @yourname:matrix.org 中的 ）、密码，并确认密码。

  5. 完成验证与登录

    • 输入恢复密钥，即前面要求保存好的一串4个英文字母组成的恢复密钥

注意事项

  • 服务器选择：若需更高隐私性，可自建服务器（请自行上网搜索教程，没有资金不建议）或选择隐私友好的公共服务器。（即不要选择国内的公共服务器）
  • 功能限制：部分实验性功能（如音视频通话）可能尚未稳定，需结合其他 Matrix 客户端使用。
  • 跨平台同步：同一账号可在多设备登录，消息实时同步。

如需进一步了解服务器搭建或高级功能配置，可参考 FluffyChat 官方 Wiki 。
